# UnscrambleHQ Architecture

UnscrambleHQ is organized into three layers:

1. UI layer: React components and pages.
2. Engine layer: search, scoring, filtering, ranking, and dictionary access.
3. Growth layer: SEO pages, metadata, analytics, and content expansion.

The engine should remain reusable across Unscrambler, Word Finder, Anagram Solver, Wordle Helper, Scrabble Helper, and Crossword Solver.
